## PHASE 6 · LAUNCH SEQUENCE


| Day                        | Action                                                                                         | What to watch                                                                                                 |
| -------------------------- | ---------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------- |
| **Day 1 morning**          | Enable the campaign.                                                                           | Status "Eligible" within 5 min. First impressions within 4 hours.                                             |
| **Day 1-3 (72-hour rule)** | **Do nothing** except the daily search-term scan below. No budget, bid, copy, or page changes. | Impression Share: if under 20% after 48 hours, raise Max CPC by $0.15 — ONCE. That's the only allowed change. |
| **Day 4-6**                | Continue the daily scan. Note per-angle CTRs; act on nothing yet.                              | The lowest-CTR angle is your weakest — flag it for Day 7, don't touch it.                                     |
| **Day 7**                  | First decision point → Phase 7.                                                                | Combined conversions ÷ total clicks.                                                                          |




### The daily 5-minute routine (every morning, Days 1-7)

1. Google Ads → Keywords → **Search Terms** → date range "Yesterday"
2. Sort by Impressions, scan the top 30-50 terms
3. Any term that isn't a genuine buyer query → add as a campaign-level negative immediately
4. Typical adds: exam names (SSC, UPSC…), course platforms (Vedantu, Khan…), "free download," "tutorial," "salary"



### Not allowed during Days 1-7

❌ Pausing keywords · ❌ Editing ad copy · ❌ Editing the page · ❌ Changing budget · ❌ More than the one $0.15 bid raise. Each of these resets Google's learning phase and restarts your 7-day clock.

## PHASE 7 · DECISION RULES — DAY 7 + 100 CLICKS

Both conditions must be true before any decision: **Day 7 AND 100 clicks.**

**Combined CVR** = (systemeclick + getresponseclick conversions) ÷ total clicks.


| Result at 100+ clicks | Decision                                                                      |
| --------------------- | ----------------------------------------------------------------------------- |
| **0 conversions**     | Diagnose first (below), fix, restart the 7-day clock once. Cut if it repeats. |
| **1 conversion**      | Continue, touch nothing, run to 200 clicks.                                   |
| **2 conversions**     | Continue with confidence. If CVR holds at 200 clicks, raise budget 50%.       |
| **3+ conversions**    | Scale — double the daily budget tomorrow.                                     |




### If 0 conversions — diagnose in this order

1. Affiliate link broken? (incognito click-through → check Systeme.io dashboard)
2. Page broken on mobile? (open on your actual phone)
3. Bid too low? (Impression Share under 30% → raise CPC $0.20, wait 48h)
4. Wrong searches triggering? (Search Terms report → add negatives)
5. Student traffic still flooding? (add 20-30 more aggressive negatives)
6. Ad disapproved? (Ads  Assets → red status → Appendix A)
7. Merchant's free-signup flow broken? (test systeme.io signup as a fresh user)



### Scale ladder

$5/day (proof) → $10 (signal confirmed, hold 7 days) → $25 (stable, hold 7 days) → $50 (mature) → add Wave 2.

### Metrics at a glance


| Metric                     | Scale ✓ | Watch ⚠ | Fix ✗            |
| -------------------------- | ------- | ------- | ---------------- |
| Combined CVR at 100 clicks | 3%+     | 1-2%    | 0%               |
| Ad CTR                     | >5%     | 2-5%    | <2%              |
| Cost per conversion        | <$15    | $15-30  | >$30             |
| Impression Share           | >50%    | 30-50%  | <30% → raise bid |

## Appendix A · If an ad gets disapproved

Common for new accounts; not a crisis. Google states the reason under Ads  Assets → red "Disapproved" label.


| Reason                            | Fix                                                                                                                                                       |
| --------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Destination mismatch              | Final URL must exactly match the live page URL — no trailing-slash or www differences.                                                                    |
| Landing page doesn't comply       | Usually the trust-page problem — which Phase 1 now prevents. Verify footer links work, privacy policy is live, page loads fast on mobile. Request review. |
| Circumventing systems (affiliate) | Thin-page flag. Your page has 500+ words, table, FAQ, verdict — protected. If flagged anyway: add 200 more words of original analysis, request review.    |
| Ad copy policy                    | Check capitalization and superlatives. The v2 ad copy doc is written to be compliant.                                                                     |
| Trademark                         | Shouldn't occur — all headlines are category-framed. If it does, Google names the offending asset; swap it.                                               |


Appeal: Ads  Assets → disapproved ad → Request Review → note that the page is genuine independent comparison content. Usually reviewed within 1 business day.
